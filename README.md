## ルール

wikipedia をご覧ください。

## main ブランチの位置づけ

CLIアプリとして、とりあえず最低限遊べる状態を実装した状態です。
よくあるGUIで遊ぶものとは以下の点が異なります。
- 縦と横が逆
- 山札からは1枚ずつめくり、最後までめくったら最初に戻る
- シーケンスの途中から別のシーケンスに移動できない（実装をさぼってるだけ）

その他、バグがたくさんあります。

## 追加したい機能例（順はおおよそ難易度が低いものから）

- シーケンスの途中から別のシーケンスに移動する
- 入力の最適化
- 終了
- ニューゲーム
- 局面の保存、保存した局面からの再開
- クリア判定
- クリアが自明な場合の自動クリア
- 積んだ時のゲームオーバー
- ヒント
- undo（一つだけ、又は、最初の状態まで）
- 新規ゲームの難易度選択
- 新規ゲームは必ずクリアできる
- クリア時間/手数ランキング
- GUI
